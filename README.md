# 🌦️ Weather Dashboard (Streamlit + MySQL)

## 📌 Overview
This project is a real-time weather dashboard that fetches live weather data using API and stores it in MySQL. The dashboard visualizes temperature, humidity, and city comparisons.

## 🚀 Features
- Real-time weather data fetching (OpenWeather API)
- MySQL database integration
- Interactive Streamlit dashboard
- KPI metrics (Temperature, Humidity, Wind Speed)
- Trend analysis with graphs
- Multi-city comparison

## 🛠️ Tech Stack
- Python
- Streamlit
- MySQL
- Pandas
- Matplotlib
